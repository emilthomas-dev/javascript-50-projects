# Project 01 – Number Guessing Game

## 📌 Description

A beginner-friendly JavaScript project where the computer generates a random number between **1 and 10**, and the user keeps guessing until the correct number is found.

## 🚀 Features

- Random number generation
- User input using `prompt()`
- Feedback:
  - Too High
  - Too Low
  - Correct
- Uses a `while` loop until the correct guess

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 JavaScript Concepts Practiced

- Variables
- Math.random()
- Math.floor()
- prompt()
- Number()
- if...else
- while loop
- console.log()
